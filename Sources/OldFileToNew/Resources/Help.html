OldFileToNew

Convert old documents to modern formats.

Drag one or more old files onto the OldFileToNew window or icon — or choose File ▸ Open… — and each is converted and saved next to the original. You can also drop folders’ worth of files at once (folders themselves are skipped).

How it works

OldFileToNew recognizes each file automatically — you don’t pick the input type. By default it writes OpenDocument files (.odt, .ods, .odp, .odg), which open in LibreOffice, Microsoft Office, Apple’s iWork apps, and most modern software.

In Settings… you can choose a more familiar output format for each kind of document. Text documents can be saved as Word, RTF, HTML, or plain text using tools built into macOS. Spreadsheets, presentations, and drawings can be saved as Excel, PowerPoint, PDF, or images if you have the free LibreOffice installed; those options are dimmed until it is. Converting to any non-OpenDocument format may lose some formatting.

Text encoding

Old files don’t record which character set they use. OldFileToNew’s default (Automatic) works for most Mac documents (Mac Roman). If accented or special characters look wrong, open Settings… and pick the matching encoding — for example MacCyrillic, MacGreek, or a Windows code page.

Supported formats

OldFileToNew converts documents from all of the applications below, spanning classic Mac software, WordPerfect, Microsoft, Apple iWork, and several drawing and page-layout programs.

Notes & limitations

  • Some very old or damaged files may not convert; the app will say so rather than write a broken file.
  • Old Mac Excel files are not currently supported.
  • Package-format iWork documents (folders) are not yet supported — export a single-file copy first.
  • Converting to a non-OpenDocument format may lose some formatting.
